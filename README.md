# practica 2 - responde las preguntas
# 1.¿Que es una Base De Datos Relacional?
   Una base de datos relacional es un tipo de base de datos que almacena y proporciona acceso a puntos de datos relacionados entre si
# 2. ¿Que es una Tabla en el contexto Base De Datos Relacionales?
   Es una estructura fundamental que se utiliza para organizar y almacenar datos en un formato estructurado
# 3. ¿Que es una relacion en el contexto de Base De Datos Relacionales?
   vinculo logico entre dos o mas tablas, establecido mediante campos comunes  que permite organizar la informacion, evitar la duplicacion de datos y garantizar la integridad referencial
# 4. ¿Que representa una columna en el contexto de Base De Datos Relacionales?
   un atributo, campo o caracteristica especifica de los datos almacenados en una tabla
# 5. ¿Que representa un renglon en el contexto de Base De Datos Relacionales?
   un registro unico, individual y estructurado de datos dentro de una tabla, tambien denominado tupla
# 6. ¿Que es un motor de base de datos relacional?
   es un software que gestiona, organiza y almacena datos en tablas estructuradas con filas y columnas, permitiendo relacionar informacion entre ellas mediante claves primarias y foraneas
# 7. ¿Cuales son los motores de bases de datos relacionales mas populares?
    Oracle Database: Lider en entornos empresariales, financieros y de alta seguridad, destacando por su gran rendimiento
    MySQL: El motor de codigo abierto mas popular, ampliamente utilizado para aplicaciones web y CMS (como WordPress)
    Microsoft SQL Server: Muy comun en entornos corporativos que utilizan tecnologias de Microsoft
    PostgreSQL: Reconocido por su robustez, funciones avanzadas, integridad de datos y ser de codigo abierto
# 8. ¿Cual es el motor de base datos ligero que no necesita un servidor?
   SQLite Funciona como una biblioteca integrada, lo que significa que la base de datos es un unico archivo en el disco local y se integra directamente en la aplicacion, siendo ideal para aplicaciones moviles, embebidos y escritorio 
# 9. ¿Que significa SQL en el contexto de base datos relacionales?
   Structured Query Language es el lenguaje estandar utilizado para gestionar, manipular y recuperar informacion en bases de datos relacionales.
# 10. ¿Cuales son los elementos de un sistema de informacion basado en base de datos relacionales?
   Sus elementos fundamentales incluyen claves primarias y foraneas para relacionar tablas, el lenguaje SQL para manipulacion, y el cumplimiento de propiedades ACID
